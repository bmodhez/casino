### 4. **Animation System**

#### Entrance Animations:
1. **Hero Logo**: Scale + opacity (0.8 → 1.0) @ 0.2s delay
2. **Headline**: Fade up @ 0.3s delay
3. **Subheading**: Fade up @ 0.4s delay
4. **Badges**: Scale + fade @ 0.6s+ (staggered)
5. **Divider**: ScaleX animation @ 0.7s delay
6. **Section Title**: Fade @ 0.8s delay
7. **Game Cards**: Staggered grid animation (0.06s each)
8. **Bottom Section**: Fade + slide up @ 0.8s delay
9. **Stat Cards**: Individual fade ups (0.1s stagger)
10. **CTA Note**: Final fade @ 1.2s delay

#### Hover Animations:
- Game cards lift 8px + scale 1.02
- Background particles scale 1.5x
- Arrow icons translate
- Icon buttons rotate 6°
- Stat cards scale 1.05x
- Gradient overlays fade in
